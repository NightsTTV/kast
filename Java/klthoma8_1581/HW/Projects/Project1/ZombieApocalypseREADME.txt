# Zombie Apocalypse Game

## Description

The Zombie Apocalypse Game is a simple text-based adventure game implemented in Java, where players navigate through levels while avoiding zombies and reaching the exit. The game features player movement, zombie AI, and a win/lose condition based on player and zombie interactions.

## Features

- **Multiple Levels:** Navigate through different levels, each with unique challenges.
- **Player Movement:** Move the player character using the keyboard (WASD keys).
- **Zombie AI:** Zombies move randomly on the grid and can catch the player.
- **Win and Lose Conditions:** Reach the exit to win; being caught by a zombie results in game over.
