This project implements recursive solutions for common computational problems, as well as two optional bonus exercises. Practicing recursion without using loops or static variables.

Run commands: (on windows)
javac RecursionUtils.java Student.java RecursionTest.java
java -cp ".;junit-4.13.2.jar;hamcrest-core-1.3.jar" org.junit.runner.JUnitCore RecursionTest

javac -cp ".;junit-4.13.2.jar;hamcrest-core-1.3.jar" BONUS1/LongMultiplication.java BONUS1/LongMultiplicationTest.java
java -cp ".;junit-4.13.2.jar;hamcrest-core-1.3.jar;BONUS1" org.junit.runner.JUnitCore LongMultiplicationTest

javac -cp ".;junit-4.13.2.jar;hamcrest-core-1.3.jar" BONUS2/*.java
java -cp ".;junit-4.13.2.jar;hamcrest-core-1.3.jar;BONUS2" org.junit.runner.JUnitCore TrainTest
