# PetiteBete Java Program  

## **1. Compilation Instructions**  
To compile the PetiteBete program, follow these steps:  

2. Open a terminal or command prompt in the directory where the `PetiteBete.java` and `Main.java` files are located.  
3. Compile both files using the following command:  javac PetiteBete.java Main.java

This will generate "PetiteBete.class" and "Main.class" files.

---

## **2. Running the Program**  
After compilation, run the program using:  java PetiteBete.java Main.java


---

## **3. Testing the Code**  
### Expected Output:  
The program will create two `PetiteBete` objects and simulate an attack.  

#### **Example Test Case:**  
- bete1 (Dragon) attacks bete` (Goblin).  
- The damage is calculated and deducted from `bete2`'s tolerance.  
- The new tolerance value of `bete2` is displayed.  

#### **Expected Console Output Example:**  
Dragon attacks Goblin for 30 damage! Goblin's new tolerance: 50


---

## **4. Modifications & Debugging**  
- You can modify the attack() method to adjust how damage is calculated.  
- `System.out.println()` statements in `takeDamage()` or `attack()` for debugging purposes.  

---

## **5. Notes**  
- The default constructor initializes a generic `PetiteBete`.  
- Modify the `Main.java` file to test different scenarios.
